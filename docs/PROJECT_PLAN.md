# 노맛동산 — 프로젝트 기획서

## 1. 목표

사용자가 **엑셀 파일**로 단어 목록을 관리하고, 웹 브라우저에서 **다양한 퀴즈 유형**으로 반복 학습할 수 있는 단어장 웹앱.

## 2. 사용자 시나리오

1. 엑셀에 단어·뜻·(선택) 약어를 정리한다.
2. 앱에서 「새 단어장」→ 엑셀 업로드 → 단어장 생성.
3. 단어장 상세에서 목록을 확인한다.
4. 「퀴즈 시작」→ 유형·문제 수 선택 → 풀이 → 결과·오답 노트 확인.

## 3. 퀴즈 유형 정의

| ID | 이름 | 출제 방식 | 필요 데이터 |
|----|------|-----------|-------------|
| `word-to-meaning` | 단어→뜻 | 4지선다 | 단어, 뜻 |
| `meaning-to-word` | 뜻→단어 | 4지선다 | 단어, 뜻 |
| `abbreviation` | 약어 | 4지선다 | 약어 열 |
| `short-answer` | 단답형 | 입력 | 단어, 뜻 |
| `mixed` | 혼합 | 위 유형 랜덤 | 단어, 뜻 (+약어 시 약어 포함) |

오답 선택지는 **같은 단어장 내 다른 항목**에서 무작위로 뽑습니다.

## 4. 데이터 모델

- **WordEntry**: id, word, meaning, abbreviation?, example?, note?
- **Wordbook**: id, name, description?, entries[], createdAt, updatedAt
- 저장: 브라우저 localStorage (`nomatdongsan-wordbooks`)

서버·회원 기능은 1차 범위에서 제외 (오프라인·개인용 우선).

## 5. 엑셀 연동 규칙

- `.xlsx`, `.xls`, `.csv` 지원
- 첫 시트만 사용
- 헤더 행 자동 매핑 (한글/영문)
- 빈 행·단어/뜻 누락 행은 스킵 후 개수 안내

## 6. 화면 구성 (1차)

| 경로 | 화면 |
|------|------|
| `/` | 홈, 기능 소개 |
| `/wordbooks` | 단어장 목록 |
| `/wordbooks/new` | 엑셀 업로드·생성 |
| `/wordbooks/:id` | 단어 목록 |
| `/wordbooks/:id/quiz` | 퀴즈 설정 |
| `/wordbooks/:id/quiz/play` | 퀴즈 진행·결과 |

## 7. 이후 단계 (우선순위)

1. **학습 통계** — 세션별 점수, 단어별 정답률
2. **오답만 복습** — 틀린 entryId 모아 재출제
3. **단어장 편집** — 앱 내 단어 추가/삭제
4. **엑셀보내기** — 수정본 다운로드
5. **배포** — GitHub Pages / Vercel

## 8. 사용자에게 필요한 것

- 엑셀 단어장 파일 (양식은 앱에서 다운로드 가능)
- 최신 Chrome / Safari / Edge 등 모던 브라우저

엑셀을 주시면 그 형식에 맞춰 헤더 매핑을 추가 조정할 수 있습니다.
